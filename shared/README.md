# SentraAI Shared Library

Contains common components, schemas, definitions, and types used across multiple services in the monorepo.

## Structure

* **`schemas/`**: JSON Schema definitions for findings, reports, and configurations.
* **`constants/`**: Severity mappings, default profiles, and static configurations.
* **`contracts/`**: Webhook schemas and API payload contracts.
* **`types/`**: Type annotations, base classes, and interfaces.
